# Snake Game

The classic Snake game written in Python. Control the snake, eat food, and try not to crash into the walls or your own tail.

## Features
- Simple and intuitive controls.
- Written in pure Python.

## How to Play
Run the `snake.py` file.

# Змейка 

Классическая игра «Змейка», написанная на Python. Управляйте змейкой, собирайте еду и старайтесь не врезаться в стены или собственный хвост.

## Особенности
- Простое и интуитивное управление.
- Написана на чистом Python.

## Как играть
Запустите файл `snake.py`.
